Установка ME системы для ComputerCraft

Всё стало проще: теперь оба файла называются ui.lua для удобства установки

Просто выполните команды для своего компьютера и наслаждайтесь мощной системой хранения!

Центральный Сервер

Установите на главном компьютере, который управляет хранилищем:

1 Подключите модем и сундуки
2 Выполните в терминале:
wget https://1ganren.github.io/ComputerCraft/ME/server/ui.lua ui.lua
3 Запустите систему:
ui

Важно!

Файл автоматически сохранится как ui.lua

Терминал доступа

1 Подключите беспроводной модем
2 Выполните в терминале:
wget https://1ganren.github.io/ComputerCraft/ME/terminal/ui.lua ui.lua
3 Запустите систему:
ui

Использование системы

1 Сброс предметов: Поместите предметы в Dump Chest
2 Получение предметов: Забирайте из Pickup Chest
3 Поиск: Вводите название предмета в интерфейсе
4 Извлечение: Нажимайте 1-9 для выбора предмета

Горячие клавиши

Tab - очистка сундуков

Enter - обновление инвентаря

Ctrl+T - экстренная остановка

Советы

• Используйте бочки для массового хранения

• Подключайте до 20 сундуков к серверу

• Advanced модемы увеличивают радиус действия

• Регулярно очищайте Dump и Pickup сундуки
